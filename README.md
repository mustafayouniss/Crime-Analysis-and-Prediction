# Crime Analysis and Prediction
A data science project that applies data mining and machine learning techniques to analyze historical crime data, discover hidden patterns, and predict potential crime trends. The project explores crime patterns over time, location distributions, and the most frequent crime categories using data analysis and visualization.

## Example Insights
- Certain crime categories appear more frequently in specific districts.
- Crime incidents show variation depending on the time of day.
- Some crime types increased during specific years.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- LightGBM

## Project Steps
1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Feature Engineering
5. Machine Learning Model Training

## Dataset
The dataset is included in a compressed (WinRAR) file. You can download or clone the repository and extract it easily. A Kaggle dataset link is also provided in the project overview presentation in this repository.
The dataset includes crime records such as date, category, district, and location coordinates.
